# Prediplay

Prediplay is a full-stack football analytics application with a web frontend and a backend API.
We gather player statistics for the top 5 major leagues via OpenAI-powered data processing, then run our own calculations and predictions to produce Prediplay scores.
Our scoring highlights:

- top players
- benchwarmers
- hidden gems
- red flags

The project is structured as a split frontend/backend repository:

- `frontend/` - Angular application (UI)
- `backend/` - Backend services and data layer

## Repository Structure

- `README.md` (this file): high-level project overview and navigation
- `frontend/README.md`: frontend-specific setup and development commands
- `backend/README.md`: backend-specific setup and development commands

## Getting Started

1. Read `frontend/README.md` for UI setup and run instructions.
2. Read `backend/README.md` for API/database setup and run instructions.

Run frontend and backend in separate terminals during development.
